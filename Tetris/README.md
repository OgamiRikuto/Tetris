# Tetris
C言語でOOP風に書いたテトリスです。
MVCモデルを使用しています。

## 実行方法
makeを使用しています。  
Tetrisプロジェクトのルートディレクトリにて次のコマンドでコンパイルされます。

```bash
$ make
```

実行は次のコマンドです。

```bash
$ make test
```

コンパイルすると、オブジェクトファイルや実行ファイルが作成されます。  
それらを削除する時は次のコマンドを打ちます。

```bash
$ make clean
```

makeをインストールしていない場合のために,シェルスクリプトも用意しています.

シェルスクリプトに実行権限を与えます.
```bash
$ chmod +x compile.sh
```
コンパイルします.
```bash
$ ./compile.sh
```

実行します.
```bash
$ ./compile.sh test
```

オブジェクトファイル等を削除します.
```bash
$ ./compile.sh clean
```

## MacOSを使用の方へ
MacOSのみ`make install`にて`.app`ファイルをで実行できます.

初回のみ
```bash
$ make install
```
でアプリケーションとして実行できます.  

なお`make clean`した場合は再度`make install`が必要になります.


## 操作方法

- 上矢印キー : ドロップ
- 下矢印キー : 下に移動
- 右矢印キー : 右に移動
- 左矢印キー : 左に移動
- zキー : 左回転
- xキー : 右回転
- spaceキー : ホールド
- escキー : 終了
- Enterキー : 決定(タイトル画面)
