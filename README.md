# Tetris
C言語でOOP風に書いたテトリスです。
MVCモデルを使用しています。

## 実行方法
makeを使用しています。  
Tetrisプロジェクトのルートディレクトリにて次のコマンドでコンパイルされます。

```bash
$ make
```

実行は次のコマンドです。

```bash
$ make test
```

コンパイルすると、オブジェクトファイルや実行ファイルが作成されます。  
それらを削除する時は次のコマンドを打ちます。

```bash
$ make clean
```

## 操作方法

- 上矢印キー : ドロップ
- 下矢印キー : 下に移動
- 右矢印キー : 右に移動
- 左矢印キー : 左に移動
- zキー : 左回転
- xキー : 右回転
- spaceキー : ホールド
- escキー : 終了
- Enterキー : 決定(タイトル画面)
